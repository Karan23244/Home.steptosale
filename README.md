# Steptosale - React + Tailwind Template

Quick scaffold: Vite + React + Tailwind with reusable components.

## Setup

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

Files of interest:
- `src/components/Header.jsx`, `Button.jsx`, `Card.jsx`, `Footer.jsx`
- `src/App.jsx` uses the components
- `tailwind.config.cjs` and `postcss.config.cjs` for Tailwind setup
